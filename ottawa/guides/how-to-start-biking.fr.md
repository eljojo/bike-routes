- cultivez la curiosité, explorez de nouvelles zones

### Rouler dans les rues
Le ministère des Transports de l'Ontario a créé un [**guide PDF sur les compétences cyclistes**](https://files.ontario.ca/mto-young-cycling-skills-en-2021-09-16.pdf), qui est un excellent aperçu de ce qu'il faut faire en roulant dehors. Il est particulièrement recommandé pour les gens qui ne conduisent pas de voiture, puisqu'il couvre les attentes des conducteurs. Le [site web sur la sécurité à vélo](https://www.ontario.ca/page/bicycle-safety) du MTO est aussi une excellente lecture!

Quelques points importants :

- Vous avez besoin de lumières (avant + arrière) et d'une clochette. Le casque est optionnel si vous avez plus de 18 ans.
- En Ontario, un vélo est considéré comme un véhicule (c'est-à-dire que vous pouvez rouler sur la plupart des routes).
- Vous ne pouvez pas rouler sur les trottoirs ni les passages piétons.
- Les voitures doivent garder un mètre de distance en vous dépassant.
