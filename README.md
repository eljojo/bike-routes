# Bike Routes Data

Structured cycling data for [Ottawa by Bike](https://ottawabybike.ca) — a curated guide to cycling routes in the National Capital Region.

This is the content repository: routes, places, guides, events, and GPX files. The [Astro app](https://github.com/eljojo/bike-app-astro) renders this data into a website.

## Structure

```
ottawa/
├── routes/       # Route content (markdown + GPX files)
├── places/       # Points of interest (cafes, parks, bike shops, etc.)
├── guides/       # Written cycling guides
├── events/       # Calendar events by year
├── organizers/   # Event organizer profiles
└── config.yml    # City configuration
```

## Contributing

Everyone is welcome to contribute routes, places, and events. The long-term vision is a "Wikipedia of bike routes" — a community-driven cycling knowledge base.
